# Edict.Azure.Persistence

Azure Table Storage projection store, Orleans grain persistence, and reminders for the [Edict](https://github.com/MalcolmMcNeely/Edict) CQRS framework.

Pair this with any Edict streaming package — `Edict.Azure.Streaming` for an all-Azure deployment, or `Edict.Kafka` for Kafka streaming with Azure state.

## Install

```
dotnet add package Edict.Azure.Persistence --prerelease
```
